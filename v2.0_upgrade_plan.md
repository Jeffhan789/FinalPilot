# FinalPilot（学呀学）v2.0 升级方案

> 制定日期：2026-07-13  
> 当前版本：v0.1 本地 MVP  
> 目标版本：v2.0 深度升级

---

## 项目现状分析

### 已实现功能
- 5 个 Tab 的完整界面：今日、计划、课程、练习、校招
- 考试倒计时、知识手卡、练习题、任务管理
- 薄弱点诊断、掌握度跟踪、错题反馈
- 双轨调度（考试复习 + 校招面试）

### 当前限制
1. **数据无持久化**：所有数据基于 `SeedData` 硬编码，App 重启后全部丢失
2. **无用户自定义**：不能添加/修改课程、知识点、考试日期
3. **无动画过渡**：界面切换生硬，缺乏现代 iOS 应用的流畅感
4. **仅支持 iPhone**：无 iPad/macOS 适配
5. **无暗色模式**：不支持 iOS 系统暗色模式切换
6. **无搜索功能**：知识手卡、题库无法搜索
7. **无数据导入/导出**：无法从其他来源导入学习资料
8. **无推送提醒**：没有复习提醒、考试倒计时通知
9. **架构待优化**：缺少依赖注入、单元测试、MVVM 严格分层
10. **无统计分析**：缺少学习趋势图表、时间线分析

## 六大升级方向

### 方向一：数据持久化与 SwiftData 迁移（Foundation）
**优先级：最高** | **工作量：高**

将纯内存数据模型迁移到 SwiftData，实现数据的持久化存储。

#### 具体任务：
- [ ] 为所有 Model 添加 `@Model` 注解：`Course`, `KnowledgePoint`, `QuizQuestion`, `StudyTask`, `CareerEvent`, `QuizAttempt`
- [ ] 设计 SwiftData 关系模型：`Course -> KnowledgePoint -> QuizQuestion` 的级联关系
- [ ] 创建 `ModelContainer` 配置，支持轻量级迁移
- [ ] 实现 `SeedData` 到 SwiftData 的初始化导入（仅首次启动）
- [ ] 重写 `FinalPilotStore`：从 `@Published` 内存数组改为 `@Query` 查询 + `modelContext` 操作
- [ ] 添加数据备份/恢复功能（导出为 JSON）
- [ ] 支持 iCloud 同步（`cloudKit` 选项）

#### 技术亮点：
- SwiftData 是 Apple 最新的数据框架，体现对现代 iOS 技术的掌握
- 支持轻量级迁移，展示数据库设计能力
- iCloud 同步展示云端架构能力

---

### 方向二：用户体验与视觉升级（UX）
**优先级：高** | **工作量：中高**

全面提升界面质感和交互体验。

#### 具体任务：
- [ ] 实现系统暗色模式自适应（`ColorScheme` 适配）
- [ ] 添加自定义主题系统：允许用户选择 3-5 种配色方案
- [ ] 为所有页面切换添加动画过渡（`.matchedGeometryEffect`, `.transition`）
- [ ] 实现知识手卡 3D 翻转效果（翻卡复习）
- [ ] 为任务完成添加 confetti 庆祝动画
- [ ] 添加 haptic feedback（触觉反馈）到按钮点击和任务完成
- [ ] 实现 Skeleton Loading 占位效果
- [ ] 优化空状态（Empty State）设计，添加插画
- [ ] 添加 App 启动引导页（Onboarding）

#### 技术亮点：
- `matchedGeometryEffect` 展示高级动画能力
- 暗色模式体现对 iOS 系统特性的深度理解
- 3D 翻转效果视觉冲击力强，适合作品集展示

### 方向三：智能功能与 AI 集成（Intelligence）
**优先级：高** | **工作量：高**

将 AI 能力深度融入学习流程，体现项目定位中的"多智能体协作"和"神经网络分析"。

#### 具体任务：
- [ ] 实现基于艾宾浩斯遗忘曲线的智能复习调度算法
- [ ] 添加答题时间分析：记录每道题答题用时，识别"犹豫题"
- [ ] 实现"Confidence Trap"检测：高自信但答错的题目自动标记为危险区域
- [ ] 添加学习趋势预测：基于历史数据预测考试分数区间
- [ ] 实现智能摘要功能：对知识手卡内容自动生成考试要点摘要
- [ ] 添加语音朗读功能：支持知识手卡 TTS 朗读
- [ ] 实现拍照导入：OCR 识别手写笔记并自动转化为知识手卡
- [ ] 集成 Kimi/OpenAI API：自动生成练习题和解释

#### 技术亮点：
- 智能调度算法体现算法设计能力
- 学习趋势预测体现数据分析和机器学习应用
- AI 集成展示对 LLM/Agent 技术的理解（契合 C310 多智能体课程背景）

扩展应用支持平台，提升使用场景。

#### 技术亮点：

- 多平台适配展示 SwiftUI 的跨平台能力

## 推荐升级路径

### 阶段一：Foundation（2-3 周）
1. SwiftData 迁移 + 数据持久化
2. 架构升级（MVVM 分层 + 依赖注入）
3. 单元测试基础覆盖

### 阶段二：UX & Platform（2 周）
1. 暗色模式 + 主题系统
2. 动画和过渡效果
3. iPad 适配 + Widget + Live Activity

### 阶段三：Intelligence（2-3 周）
1. 智能复习调度算法
2. 答题分析 + 学习趋势预测
3. AI 集成功能（如 TTS、智能摘要）

### 阶段四：Integration & Polish（1-2 周）
1. 搜索 + 数据导入导出
2. 日历集成 + 分享
3. 性能优化 + 文档完善

**总计：约 7-10 周完整迭代**
