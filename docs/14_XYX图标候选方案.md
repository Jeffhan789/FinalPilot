# 14 XYX 图标候选方案

文档版本：v0.1  
日期：2026-05-01

## 1. 设计目标

本轮图标重做的目标是解决上一版 `XYX` 图标过于复杂、观感不够精致的问题。

新方向：

- 将 `XYX` 解释为“学呀学”。
- 尝试简体 `学`、繁体 `學` 与 `XYX` 的融合。
- 减少小字和复杂装饰，保证手机桌面小尺寸可识别。
- 保留 FinalPilot 的复习调度、课程路线、云计算和神经网络课程气质。

## 2. 生成结果

候选稿输出目录：

- `design/icon_shortlist/`
- `design/icon_candidates/`

重点审阅第二轮 shortlist：

- `design/icon_shortlist/FinalPilot_XYX_shortlist.png`

第二轮候选：

| 编号 | 名称 | 设计方向 | 初步判断 |
| --- | --- | --- | --- |
| 01 | 简体极简 | 圆形徽章、简体 `学`、底部 `XYX` | 稳重，最像成熟学习工具 |
| 02 | 繁体极简 | 繁体 `學`、深色卡片、浅色背景 | 有文化感，但小尺寸稍复杂 |
| 03 | X学X | 字母和汉字直接融合 | 记忆点强，最符合“XYX 学呀学” |
| 04 | X學X | 繁体字与学习路径融合 | 有课程路线感，但主体偏细 |
| 05 | 学印章 | 中文印章式品牌符号 | 有中文辨识度，适合讲品牌概念 |
| 06 | 学书页 | 书页 + 学 + XYX | 复习感最强，但图形元素较多 |
| 07 | 云端学 | 云端节点 + 学 + XYX | 云计算课程映射明显 |
| 08 | XYX调度 | Y 代表分流调度 | 多智能体调度含义强，但视觉不如 01/03 直接 |

## 3. 推荐优先级

推荐优先审阅：

1. `01 简体极简`：最稳妥，适合正式 App。
2. `03 X学X`：最有记忆点，适合校招项目展示。
3. `05 学印章`：中文品牌感强，适合强调“学呀学”。

如果要做最终版，我建议从 `03 X学X` 继续精修。原因是它把 `X`、`Y`、`X` 和 `学` 融合在一个主视觉里，比单独放一个汉字更像项目品牌。

## 4. 后续落地

用户选定编号后，执行：

1. 将选中候选图标扩展成 iOS 所需全套 AppIcon 尺寸。
2. 替换 `FinalPilotApp/Assets.xcassets/AppIcon.appiconset/`。
3. 运行 Xcode 构建验证。
4. 更新图标说明文档和进展日志。
5. 同步到 GitHub。
