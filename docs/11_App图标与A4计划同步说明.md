# 11 App 图标与 A4 计划同步说明

文档版本：v0.2  
日期：2026-05-01  
来源 PDF：本机 `A4-两周复习进度规划表.pdf`

## 1. 图标设计

图标主题：FinalPilot 冲刺导航 + XYX 学呀学。

视觉元素：

- `XYX` 主标识：代表“学呀学”，形成更容易记住的项目品牌。
- `学呀学` 中文小字：解释 XYX 的含义，增强中文场景识别度。
- 深青色到深蓝色背景：代表学习仪表盘、云端技术和冷静复习状态。
- 橙色学习曲线：代表 5/13 C310、5/14 E320、5/26 C315 的考试节点。
- 三个节点：代表 C310、E320、C315 三门课，也表示输入、调度、输出三个学习环节。
- 打开的书页：代表复习资料、知识整理和考前输出。

落地文件：

- `FinalPilotApp/Assets.xcassets/AppIcon.appiconset/`
- `tools/generate_finalpilot_icon.swift`

说明：当前图标采用确定性绘制脚本生成，便于后续调整颜色、节点、线条和尺寸。

面试讲法：

`XYX` 可以解释为“学呀学”：左侧 `X` 是吸收资料，中间 `Y` 是计划分流与优先级判断，右侧 `X` 是练习输出。它和 App 的多智能体调度逻辑对应：复习资料进入系统，PlannerAgent / ConflictAgent 做取舍，最后输出今日任务、错题复盘和面试准备。

## 3. 手机上怎么用

在 Xcode 中运行到模拟器或真机后：

1. 打开 FinalPilot。
2. 点击底部 `计划`。
3. 顶部查看三门考试日期：C310 5/13、E320 5/14、C315 5/26。
4. 使用阶段筛选查看 `搭骨架`、`补高频`、`真题闭环`、`考试切换`。
5. 每天按卡片完成 C310、E320、输出物和勾选项。
6. 如果当天有面试或状态差，执行 `每日保底`，不要补偿性熬夜。
