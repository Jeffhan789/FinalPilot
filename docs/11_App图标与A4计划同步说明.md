# 11 App 图标与 A4 计划同步说明

文档版本：v0.3  
日期：2026-05-01  
来源 PDF：本机 `A4-两周复习进度规划表.pdf`

## 1. 图标设计

图标主题：手绘 `XYX + 学` 参考图提取。

视觉元素：

- `XYX`：来自用户手绘稿，含义为“学呀学”。
- `学`：使用手绘笔迹提取，而不是系统字体，保留个人项目辨识度。
- 白色背景：采用接近常规软件图标的干净白色，避免使用照片里的发黄纸色。
- 蓝色主体：从手绘图中提取马克笔蓝，当前提取值为 `#265CA3`。
- 轻微阴影：只用于让蓝色笔迹在白底上更清晰，不做复杂装饰。

落地文件：

- `FinalPilotApp/Assets.xcassets/AppIcon.appiconset/`
- `tools/generate_icon_from_reference.py`
- `design/reference_icon/FinalPilot_reference_extracted_icon.png`

说明：当前图标由参考图提取脚本生成。脚本运行时读取本机参考图，但仓库不提交原始照片，只提交处理后的 AppIcon 资产和预览图。

面试讲法：

`XYX` 可以解释为“学呀学”。这版图标直接来自手绘草图，表达这个项目不是模板化 App，而是围绕真实考试压力、真实复习资料和真实校招展示做出来的个人学习工具。蓝色保留手写稿的能量，白底让它更像正式 iOS 软件图标。

## 3. 手机上怎么用

在 Xcode 中运行到模拟器或真机后：

1. 打开 FinalPilot。
2. 点击底部 `计划`。
3. 顶部查看三门考试日期：C310 5/13、E320 5/14、C315 5/26。
4. 使用阶段筛选查看 `搭骨架`、`补高频`、`真题闭环`、`考试切换`。
5. 每天按卡片完成 C310、E320、输出物和勾选项。
6. 如果当天有面试或状态差，执行 `每日保底`，不要补偿性熬夜。
